LISTA DE EXERCÍCIOS

1. Exiba um alerta com a mensagem "Boas-vindas ao nosso site!".

2. Declare as variáveis nome, idade, numeroDeVendas e saldoDisponivel com valores iniciais de sua escolha.

3. Crie uma variável mensagemDeErro com o texto "Erro! Preencha todos os campos" e exiba-a em um alerta.

4. Use um prompt para perguntar o nome do usuário e armazene-o em uma variável.

5. Peça ao usuário sua idade via prompt e armazene-a na variável idade.

6. Caso a idade digitada no exercício anterior seja maior ou igual a 18, exiba um alerta com a mensagem "Pode tirar a habilitação!".

7. Pergunte ao usuário qual é o dia da semana. Se a resposta for "Sábado" ou "Domingo", mostre "Bom fim de semana!", caso contrário, mostre "Boa semana!".

8. Peça um número ao usuário e mostre um alerta informando se o número digitado é positivo ou negativo.

9. Crie um sistema que pergunte ao usuário quantos pontos ele alcançou em um jogo: se a pontuação for maior ou igual a 100, mostre "Parabéns, você venceu!", caso contrário, mostre "Tente novamente para ganhar.".

10. Use uma template string para informar o usuário sobre o saldo da conta, incluindo o valor de uma variável em um alerta ou no console.

11. Peça o nome do usuário via prompt e mostre um alerta de boas-vindas personalizado (ex: "Bem-vindo, [nome]!").

12. Use um loop while para exibir os números de 1 a 10 no console do navegador.

13. Use um loop while para exibir os números de 10 a 0 no console do navegador.

14. Crie um programa de contagem regressiva: peça um número ao usuário e conte dele até 0 no console usando um loop.

15. Crie um programa de contagem progressiva: peça um número ao usuário e conte de 0 até esse número no console.

16. Pergunte ao usuário qual a linguagem de programação que ele mais gosta usando prompt e armazene a resposta em uma variável para exibir no console.

17. Crie duas variáveis numéricas (valor1 e valor2), realize a soma delas e exiba a mensagem: "A soma de [valor1] e [valor2] é igual a [resultado]." no console.

18. Peça a idade do usuário e utilize um if para exibir no console se a pessoa é maior ou menor de idade.

19. Utilize o console.log e o Math.random para gerar e exibir qualquer número decimal aleatório entre 0 e 1.

20. Use Math.random junto com o parseInt para gerar um número inteiro entre 1 e 10 e exiba esse número no console.
